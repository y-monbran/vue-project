# README

## アプリ名:  
タングラムパズル（偽物）

## 概要:  
枠内に四角形をドラッグ＆ドロップできる。

## 本番環境:  

表示されるまで時間がかかります。30秒ほどお見積もりいただけると幸いです。  

## 制作背景(意図):  
JavaScriptを勉強したいと思いました。その際プログラミングスクールで習わなかったVue.jsをやろうと思いました。
